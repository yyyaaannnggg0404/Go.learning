## 使用var关键字
* var a, b, c bool
* var s1, s2 string = "hello", "world"
* 可放在函数内，或直接放在包内
* 使用var()集中定义变量

## 让编译器自动决定类型
* var a, b, i, s1, s2 = true, false, 3, "hello", "world"

## 使用:=定义变量
* a, b, i, s1, s2 := true, false, 3, "hello", "world"
* 只能在函数内使用

```
var text = "Hello, world!"
text := "Hello, world!"
var (
  text1 = "Hello"
  text2 = ","
  text3 = "world"
  text4 = "!"
)
```
