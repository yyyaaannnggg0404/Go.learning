# Slice（切片）
```
arr := [...]int{0, 1, 2, 3, 4, 5, 6, 7}
s := arr[2:6]
```
* s的值为[2 3 4 5]
```
arr := [...]int{0, 1, 2, 3, 4, 5, 6, 7}
s := arr[2:6]
s[0] = 10
```
* Slice本身是没有数据，是对底层array的一个view
* arr的值变为[0 1 10 3 4 5 6 7]
## Reslice
```
s := arr[2:6]
s = s[:3]
s = s[1:]
s = arr[:]
```
## Slice的扩展
```
arr := [...]int{0, 1, 2, 3, 4, 5, 6, 7}
s1 := arr[2:6]
s2 := s1[3:5]
```
* s1的值为[2 3 4 5]，s2的值为[5 6]
* slice可以向后扩展，不可以向前扩展
* s[i]不可以超越len(s)，向后扩展不可以超越底层数组cap(s)
## 向Slice添加元素
```
arr := [...]int{0, 1, 2, 3, 4, 5, 6, 7}
s1 := arr[2:6]
s2 := s1[3:5]
s3 := append(s2, 10)
s4 := append(s3, 11)
s5 := append(s4, 12)
```
* 添加元素时如果超过cap，系统会重新分配更大的底层数组
* 由于值传递的关系，必须接手append的返回值
* s = append(s, val)
